pyControl is a system of software and hardware for controlling behavioural experiments based around the Micropython microcontroller.

For more information please see the Docs: http://pycontrol.readthedocs.io

Repository contents:

CLI      : The pyControl command line interface.
config   : CLI Configuration files.
data     : Location where data generated by the CLI is stored.
devices  : pyControl hardware classes (used on the pyboard).
pyControl: The pyControl framework (used on the pyboard).
tasks    : pyControl task files.
licence  : pyControl-GPL-3 licence.

Version: v1.1.3
